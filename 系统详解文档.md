## 1. 核心游戏循环系统

### 1.1 系统概述
核心游戏循环是整个游戏的骨架，定义了玩家的主要游戏行为和目标。

### 1.2 大循环（Meta Loop）

```
┌─────────────────────────────────────────────────────┐
│                  【公会阶段】                         │
│  管理猎手、整理仓库、升级设施、交易买卖              │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│                  【准备阶段】                         │
│  选择猎手 → 配置装备 → 选择区域 → 出发              │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│                  【探索阶段】                         │
│  搜刮物资 → 战斗怪物 → 寻找撤离点                   │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│                  【结算阶段】                         │
│  成功撤离：物资+经验 / 失败：损失物资+猎手状态变化   │
└─────────────────────────────────────────────────────┘
                         ↓
                    返回公会阶段
```

### 1.3 单局循环（搜-打-撤）

| 阶段 | 行为 | 目标 | 风险 |
|------|------|------|------|
| **搜** | 探索地图、开箱搜刮 | 获取物资 | 时间消耗、遭遇敌人 |
| **打** | 与怪物战斗 | 击杀获取掉落 | 消耗资源、可能死亡 |
| **撤** | 前往撤离点撤离 | 安全带走物资 | 撤离过程可能被打断 |

### 1.4 设计目的
- 创造"每一局都有意义"的体验
- 平衡风险与收益
- 让玩家不断面临决策

## 2. 职业系统

### 2.1 系统概述
职业系统决定了猎手的战斗风格、技能和成长方向。每个职业都有独特的定位和玩法。

### 2.2 职业列表

| 职业 | 定位 | 难度 | 主属性 | 特点 |
|------|------|------|--------|------|
| 剑士 | 近战平衡 | ★★☆ | 力量/体质 | 攻守兼备，适合新手 |
| 狂战士 | 近战爆发 | ★★★ | 力量 | 高伤害高风险 |
| 盗贼 | 刺客/敏捷 | ★★★ | 敏捷 | 暴击背刺，高操作 |
| 游侠 | 远程物理 | ★★☆ | 敏捷/感知 | 安全输出，控制 |
| 法师 | 远程魔法 | ★★☆ | 智力 | AOE伤害，元素掌控 |
| 牧师 | 辅助生存 | ★☆☆ | 智力/体质 | 治疗护盾，高生存 |
| 召唤师 | 召唤流 | ★★★ | 智力 | 召唤生物作战 |
| 炼金术士 | 道具流 | ★★★ | 智力/敏捷 | 药剂陷阱，战术多样 |

### 2.3 职业特殊机制

| 职业 | 特殊机制 | 说明 |
|------|----------|------|
| 狂战士 | 狂怒值 | 受伤/攻击积累，增加伤害但也增加受到的伤害 |
| 盗贼 | 潜行/背刺 | 脱战后可潜行，背后攻击有加成 |
| 游侠 | 蓄力/标记 | 蓄力增加伤害，标记敌人增加攻击加成 |
| 法师 | 元素切换 | 可切换火/冰/雷元素 |
| 牧师 | 神圣能量 | 治疗消耗神圣能量，随时间恢复 |

### 2.4 职业进阶
- 达到一定等级可解锁进阶职业
- 示例：剑士 → 剑圣 / 圣骑士

### 2.5 设计目的
- 提供多样化的战斗体验
- 满足不同玩家的偏好
- 增加游戏深度和重玩价值

## 3. 猎手管理系统

### 3.1 系统概述
猎手管理系统允许玩家招募、培养和管理多个猎手角色。

### 3.2 猎手属性

```
猎手信息
├── 基础信息
│   ├── 名字
│   ├── 职业
│   ├── 品质（白/蓝/紫/金）
│   └── 等级
├── 属性值
│   ├── 力量、敏捷、智力
│   ├── 体质、精神、幸运
│   └── 生命值、魔法值
├── 技能
│   ├── 已学习技能
│   └── 技能熟练度
└── 装备
    ├── 武器、防具、饰品
    └── 背包
```

### 3.3 猎手品质

| 品质 | 颜色 | 初始属性 | 成长潜力 | 获取难度 |
|------|------|----------|----------|----------|
| 普通 | ⚪白 | 低 | 低 | 容易 |
| 精良 | 🔵蓝 | 中 | 中 | 一般 |
| 稀有 | 🟣紫 | 高 | 高 | 困难 |
| 传说 | 🟡金 | 极高 | 极高 | 极难 |

### 3.4 招募方式

| 方式 | 消耗 | 品质范围 | 说明 |
|------|------|----------|------|
| 普通招募 | 金币 | 白-蓝 | 随机职业和属性 |
| 高级招募 | 稀有材料 | 蓝-紫 | 较高概率获得好品质 |
| 特殊招募 | 任务/活动 | 紫-金 | 固定或保底获得 |

### 3.5 猎手状态

| 状态 | 说明 | 处理方式 |
|------|------|----------|
| 正常 | 可以出击 | - |
| 疲劳 | 需要休息 | 等待恢复/使用道具 |
| 受伤 | 属性降低 | 医疗室治疗 |
| 重伤 | 无法出击 | 长时间休养 |
| 死亡 | 无法使用 | 复活或永久失去 |

### 3.6 设计目的
- 增加角色管理的策略性
- 建立玩家与角色的情感连接
- 提供长期养成目标

## 4. 战斗系统

### 4.1 系统概述
战斗系统是ARPG的核心，提供即时的动作战斗体验。

### 4.2 基础操作

| 操作 | 按键（示例） | 说明 |
|------|--------------|------|
| 普通攻击 | 鼠标左键 | 连续点击进行连招 |
| 技能 | 数字键1-4 | 消耗MP/有冷却时间 |
| 闪避 | 空格键 | 消耗耐力，有无敌帧 |
| 格挡 | 鼠标右键 | 减少受到的伤害（部分职业） |
| 冲刺 | Shift | 快速移动 |
| 使用道具 | F | 使用背包中的消耗品 |

### 4.3 伤害计算

```
物理伤害公式：
最终伤害 = (攻击力 - 防御力) × 技能倍率 × 暴击倍率 × 属性克制

魔法伤害公式：
最终伤害 = (魔攻 × 技能倍率) × (1 - 魔抗%) × 暴击倍率

暴击计算：
暴击率 = 基础5% + 敏捷/200 + 装备加成
暴击伤害 = 150% + 幸运/100 + 装备加成
```

### 4.4 战斗节奏设计

| 阶段 | 时长 | 特点 |
|------|------|------|
| 接敌 | 1-2秒 | 识别敌人，选择应对 |
| 交战 | 5-30秒 | 攻击、躲避、使用技能 |
| 结束 | 1-2秒 | 敌人死亡，搜刮掉落 |

### 4.5 设计目的
- 提供爽快的打击感
- 平衡操作与策略
- 让不同职业有不同的战斗体验

## 5. 撤离系统

### 5.1 系统概述
撤离系统是"搜打撤"玩法的核心，决定了玩家能否安全带走战利品。

### 5.2 撤离点类型

| 类型 | 位置 | 特点 | 风险 |
|------|------|------|------|
| 固定撤离点 | 地图固定位置 | 稳定可靠 | 可能被怪物占据 |
| 随机撤离点 | 每局随机生成 | 需要探索发现 | 位置不确定 |
| 付费撤离点 | 特定位置 | 需要消耗物资/金币 | 消耗资源 |
| 紧急撤离 | 任意位置 | 立即撤离 | 丢弃背包物资 |

### 5.3 撤离流程

```
1. 发现/到达撤离点
       ↓
2. 触发撤离（开始倒计时）
       ↓
3. 等待期间（10-30秒）
   - 可能遭遇敌人攻击
   - 被攻击会打断撤离
   - 需要边战斗边等待
       ↓
4. 倒计时结束
       ↓
5. 成功撤离！
```

### 5.4 撤离决策

| 情况 | 建议决策 |
|------|----------|
| 背包满了 | 立即撤离 |
| 血量低/药水不足 | 尽快撤离 |
| 侵蚀度高 | 必须撤离 |
| 发现高价值区域 | 权衡风险后决定 |
| 遇到精英怪 | 绕开或打完再撤 |

### 5.5 设计目的
- 创造紧张刺激的体验
- 让玩家面临"贪婪与安全"的抉择
- 增加每局游戏的戏剧性

## 6. 物资与装备系统

### 6.1 系统概述
物资与装备系统管理游戏中所有可获取、可使用的物品。

### 6.2 物资分类

| 类别 | 说明 | 示例 | 用途 |
|------|------|------|------|
| 武器 | 战斗用具 | 剑、弓、法杖 | 装备使用 |
| 防具 | 护甲装备 | 头盔、胸甲 | 装备使用 |
| 饰品 | 戒指项链 | 力量戒指 | 装备使用 |
| 消耗品 | 一次性物品 | 药水、卷轴 | 战斗中使用 |
| 材料 | 制作原料 | 矿石、皮革 | 制作/升级 |
| 贵重品 | 高价值物品 | 宝石、古董 | 出售赚钱 |
| 钥匙 | 开启特殊区域 | 地牢钥匙 | 解锁内容 |
| 任务物品 | 任务相关 | 收集品 | 完成任务 |

### 6.3 装备品质

| 品质 | 颜色 | 词缀数量 | 说明 |
|------|------|----------|------|
| 普通 | ⚪白 | 0 | 基础属性 |
| 魔法 | 🟢绿 | 1 | 1个随机词缀 |
| 稀有 | 🔵蓝 | 2 | 2个随机词缀 |
| 史诗 | 🟣紫 | 3+ | 3个词缀+特殊效果 |
| 传说 | 🟡金 | 固定 | 固定强力效果，可能有套装 |
| 神话 | 🔴红 | 唯一 | 独一无二的顶级装备 |

### 6.4 词缀系统

```
前缀示例：
├── 锋利的 → +攻击力
├── 坚固的 → +防御力
├── 迅捷的 → +攻击速度
└── 强力的 → +暴击伤害

后缀示例：
├── of Fire → 附加火焰伤害
├── of Vitality → +最大生命值
├── of the Bear → +力量
└── of Evasion → +闪避率
```

### 6.5 物资风险

```
出击携带的装备/物资
        ↓
    进入深渊
        ↓
   ┌────┴────┐
   ↓         ↓
成功撤离   猎手死亡
   ↓         ↓
物资保留   全部丢失
```

### 6.6 设计目的
- 提供装备追求的动力
- 创造物资管理的策略性
- 让每次撤离都有"赌博"的刺激感

## 7. 背包与负重系统

### 7.1 系统概述
背包系统管理猎手在探索中能携带的物资数量。

### 7.2 背包等级

| 等级 | 名称 | 格子数 | 获取方式 |
|------|------|--------|----------|
| 1级 | 布制背包 | 12格 | 初始装备 |
| 2级 | 皮革背包 | 20格 | 制作/购买 |
| 3级 | 冒险者背包 | 28格 | 稀有掉落 |
| 4级 | 次元袋 | 40格 | 传说物品 |

### 7.3 物品占格

| 物品类型 | 占用格子 |
|----------|----------|
| 小型物品（药水、材料） | 1格 |
| 中型物品（饰品、卷轴） | 2格 |
| 大型物品（防具） | 3格 |
| 武器 | 4格 |

### 7.4 负重系统

```
负重影响：
├── 0-50%：正常移动
├── 50-75%：移动速度-10%
├── 75-100%：移动速度-25%，无法冲刺
└── 100%以上：无法移动，必须丢弃物品
```

### 7.5 设计目的
- 增加资源管理决策
- 平衡搜刮与战斗
- 让"带什么"成为策略

## 8. 深渊区域系统

### 8.1 系统概述
深渊区域系统定义了游戏中可探索的各个地图区域。

### 8.2 区域列表

| 区域 | 难度 | 环境 | 主要怪物 | 主要掉落 |
|------|------|------|----------|----------|
| 遗忘墓穴 | ★☆☆ | 地下墓室 | 亡灵系 | 基础材料 |
| 腐朽森林 | ★★☆ | 黑暗森林 | 野兽/植物 | 皮革、草药 |
| 废弃矿坑 | ★★☆ | 矿洞 | 哥布林 | 矿石 |
| 沉沦神殿 | ★★★ | 古代遗迹 | 魔像 | 魔法装备 |
| 恶魔裂隙 | ★★★★ | 地狱入口 | 恶魔 | 史诗装备 |
| 深渊之心 | ★★★★★ | 终极区域 | Boss | 传说装备 |

### 8.3 区域解锁

```
遗忘墓穴（初始解锁）
    ↓ 完成主线任务
腐朽森林 / 废弃矿坑
    ↓ 完成主线任务
沉沦神殿
    ↓ 获得钥匙
恶魔裂隙
    ↓ 击败前置Boss
深渊之心
```

### 8.4 地图结构

```
区域地图构成：
├── 入口区：相对安全，物资较少
├── 中央区：主要探索区域
├── 高价值区：需要钥匙，物资丰富但敌人强
├── Boss区：Boss战区域
└── 撤离点：分布在地图各处
```

### 8.5 设计目的
- 提供渐进式难度曲线
- 让玩家有探索新区域的动力
- 不同区域提供不同体验

## 9. 深渊侵蚀系统

### 9.1 系统概述
深渊侵蚀是限制玩家探索时间的核心机制，增加紧迫感。

### 9.2 侵蚀机制

```
侵蚀度变化：
├── 进入深渊：侵蚀度 = 0%
├── 时间流逝：每分钟 +5%
├── 进入特定区域：立即 +10-20%
└── 被特定怪物攻击：+5-10%

侵蚀效果：
├── 0-25%：无影响
├── 25-50%：视野范围减少
├── 50-75%：属性开始下降（-10%）
├── 75-100%：持续掉血
└── 100%：即死
```

### 9.3 对抗侵蚀

| 方式 | 效果 | 获取 |
|------|------|------|
| 净化药水 | -20%侵蚀 | 制作/搜刮 |
| 篝火休息点 | 重置为0% | 地图固定点 |
| 神圣结界 | 暂停侵蚀 | 牧师技能 |
| 撤离 | 完全清除 | 撤离点 |

### 9.4 设计目的
- 限制无限探索
- 创造紧迫感
- 增加药水等资源的价值
- 让"撤离时机"成为重要决策

## 10. 公会系统

### 10.1 系统概述
公会是玩家的"家"，所有非战斗活动都在这里进行。

### 10.2 公会设施

| 设施 | 功能 | 初始状态 |
|------|------|----------|
| 大厅 | 猎手管理、出击准备 | 已解锁 |
| 仓库 | 存储物资 | 已解锁（50格） |
| 招募所 | 招募新猎手 | 已解锁 |
| 铁匠铺 | 武器防具制作/强化 | 需解锁 |
| 炼金室 | 药水/卷轴制作 | 需解锁 |
| 医疗室 | 治疗/复活猎手 | 需解锁 |
| 训练场 | 猎手训练、技能练习 | 需解锁 |
| 商店 | 买卖物资 | 已解锁 |
| 任务板 | 查看/接取任务 | 已解锁 |
| 图书馆 | 图鉴、学习配方 | 需解锁 |

### 10.3 公会等级

| 等级 | 解锁内容 |
|------|----------|
| 1级 | 基础设施 |
| 2级 | 铁匠铺、炼金室 |
| 3级 | 医疗室、仓库扩容 |
| 4级 | 训练场、图书馆 |
| 5级 | 高级设施功能 |

### 10.4 设计目的
- 提供"家"的归属感
- 集中所有非战斗功能
- 让玩家有建设目标

---

## 11. 设施升级系统

### 11.1 系统概述
设施升级系统让玩家逐步增强公会的功能。

### 11.2 升级详情

| 设施 | 等级 | 效果 | 所需材料 |
|------|------|------|----------|
| 仓库 | 1→2 | 容量+50格 | 木材x20、铁锭x10 |
| 仓库 | 2→3 | 容量+80格 | 钢材x30、魔法石x5 |
| 铁匠铺 | 1→2 | 解锁蓝色装备制作 | 铁锭x30、煤炭x20 |
| 铁匠铺 | 2→3 | 解锁紫色装备制作 | 钢材x50、熔岩核心x3 |
| 炼金室 | 1→2 | 解锁高级药水 | 草药x50、精华x5 |
| 医疗室 | 1→2 | 降低复活费用30% | 绷带x100、圣水x10 |
| 训练场 | 1→2 | 训练效率+50% | 木材x40、铁锭x20 |

### 11.3 升级流程

```
1. 收集所需材料
2. 在设施界面选择升级
3. 等待升级时间（可用道具加速）
4. 升级完成，解锁新功能
```

### 11.4 设计目的
- 提供长期目标
- 让探索获得的材料有用途
- 渐进式解锁内容

## 12. 商店与交易系统

### 12.1 系统概述
商店系统允许玩家买卖物资，是经济系统的重要组成。

### 12.2 商人类型

| 商人 | 销售物品 | 解锁条件 | 特点 |
|------|----------|----------|------|
| 武器商 | 基础武器 | 初始 | 稳定库存 |
| 防具商 | 基础防具 | 初始 | 稳定库存 |
| 药剂师 | 药水、卷轴 | 炼金室Lv1 | 可补充消耗品 |
| 神秘商人 | 稀有物品 | 随机出现 | 高价但物品好 |
| 黑市商人 | 特殊装备 | 声望要求 | 需要特殊货币 |

### 12.3 好感度系统

```
商人好感度：
├── 0-100：普通级商品
├── 100-300：解锁中级商品
├── 300-600：解锁高级商品
└── 600+：解锁特殊商品

提升方式：
├── 完成商人委托任务
├── 在该商人处购买物品
└── 特殊事件
```

### 12.4 设计目的
- 提供物资流通渠道
- 增加与NPC的互动
- 让"搜刮无用物品"也有价值

## 13. 任务系统

### 13.1 系统概述
任务系统引导玩家游戏流程，提供奖励和目标。

### 13.2 任务类型

| 类型 | 刷新 | 难度 | 奖励级别 |
|------|------|------|----------|
| 主线任务 | 固定 | 中等 | 高（解锁内容） |
| 日常委托 | 每日 | 简单 | 低（金币/材料） |
| 周常挑战 | 每周 | 较难 | 中（稀有材料） |
| 商人订单 | 接取后 | 不定 | 好感度+物品 |
| 猎手专属 | 固定 | 困难 | 职业专属奖励 |
| 成就任务 | 固定 | 长期 | 称号/特殊奖励 |

### 13.3 任务目标示例

| 目标类型 | 示例 |
|----------|------|
| 击杀类 | 击杀30只骷髅 |
| 收集类 | 收集10块精铁矿 |
| 探索类 | 首次到达沉沦神殿 |
| 撤离类 | 成功从恶魔裂隙撤离 |
| Boss类 | 击败骷髅王 |
| 限制类 | 不使用药水完成关卡 |

### 13.4 设计目的
- 引导玩家游戏
- 提供短期/长期目标
- 奖励多样化玩法

| 机械系 | 防御力高 | 雷电 | 神殿、遗迹 |

## 15. 属性与数值系统

### 15.2 基础属性

## 16. 技能系统

### 16.1 系统概述
技能系统定义了角色可使用的主动和被动能力。

### 16.2 技能类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 主动技能 | 需要释放 | 火球术、剑刃风暴 |
| 被动技能 | 自动生效 | 暴击率+5%、生命+10% |
| 终极技能 | 强力技能，长CD | 陨石坠落、狂暴化 |

### 16.3 技能槽位

```
技能配置：
├── 主动技能槽：4个（可热键释放）
├── 被动技能槽：3个（自动生效）
└── 终极技能槽：1个（特殊条件触发）
```

### 16.4 技能升级

```
技能升级方式：
├── 使用升级：使用技能积累熟练度
├── 技能点升级：消耗技能点直接升级
└── 技能书学习：获得新技能

技能等级效果：
├── 等级1：基础效果
├── 等级2-4：数值提升
├── 等级5：技能进化（改变效果）
```

### 16.5 设计目的
- 提供战斗深度
- 让玩家有养成目标
- 支持不同Build

## 19. 死亡与复活系统

### 19.1 系统概述
死亡系统是"高风险"体验的核心，定义了猎手死亡后的处理。

### 19.2 死亡后果

```
猎手死亡
    ↓
┌───────────────────────────────┐
│  1. 携带物资全部丢失           │
│  2. 装备丢失（除绑定装备外）   │
│  3. 猎手进入"死亡"状态        │
└───────────────────────────────┘
    ↓
处理选择
    ├── 永久死亡（困难模式）
    │       └── 猎手永久消失
    ├── 复活（消耗资源）
    │       ├── 消耗：复活药剂 + 大量金币
    │       ├── 惩罚：等级-1，需要休养
    │       └── 装备仍然丢失
    └── 灵魂保存（特殊道具）
            └── 保留猎手，但等级降低更多
```

### 19.3 复活费用

| 猎手等级 | 复活费用（金币） | 额外消耗 |
|----------|------------------|----------|
| 1-10 | 500 | 复活药剂x1 |
| 11-20 | 2000 | 复活药剂x2 |
| 21-30 | 5000 | 高级复活药剂x1 |
| 31+ | 10000+ | 高级复活药剂x2 |

### 19.4 设计目的
- 让死亡有真实的代价
- 建立与角色的情感连接
- 增加游戏紧张感

## 20. 经济系统

### 20.1 系统概述
经济系统管理游戏中的货币流通和价值平衡。

### 20.2 货币类型

| 货币 | 获取方式 | 主要用途 |
|------|----------|----------|
| 金币 | 出售物资、任务 | 购买物资、复活、招募 |
| 灵魂碎片 | 击杀怪物 | 技能升级、猎手强化 |
| 公会徽章 | 主线任务、成就 | 解锁高级内容 |

### 20.3 经济循环

```
收入来源：
├── 出售搜刮物资
├── 完成任务奖励
├── 击杀怪物掉落
└── 出售多余装备

支出项目：
├── 购买装备/消耗品
├── 设施升级
├── 猎手招募
├── 复活猎手
└── 装备修复
```

### 20.4 设计目的
- 提供资源追求动力
- 平衡游戏难度
- 让"发财"成为目标

## 22. 存档系统

### 22.1 系统概述
存档系统管理玩家的游戏进度保存。

```
存档数据：
```
