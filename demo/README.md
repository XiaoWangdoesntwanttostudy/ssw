# 深渊猎手 Demo

这是《深渊猎手》游戏的可玩Demo版本。

## 🎮 如何运行

### 方法一：直接打开
双击 `index.html` 文件在浏览器中打开即可游玩。

### 方法二：本地服务器（推荐）
如果遇到跨域问题，可以使用本地服务器：

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve .

# 或使用 VS Code Live Server 插件
```

然后访问 `http://localhost:8080`

## 🕹️ 操作说明

### 探索阶段
- **WASD** - 移动
- **鼠标左键** - 普通攻击
- **空格** - 闪避
- **Q** - 技能1（攻击）
- **E** - 技能2（闪避）
- **R** - 技能3（范围大招，消耗30MP）
- **1** - 使用生命药水
- **2** - 使用魔力药水
- **F** - 在撤离点开始撤离
- **Tab** - 打开背包
- **ESC** - 暂停菜单

## 🎯 游戏流程

1. **公会界面** - 管理猎手、购买物品
2. **出击准备** - 选择猎手 → 选择区域 → 确认出击
3. **深渊探索** - 击杀敌人、收集物资、注意侵蚀度
4. **撤离** - 到达绿色撤离点按F撤离
5. **结算** - 成功撤离保留物品，死亡失去一切

## ⚠️ 注意事项

- **侵蚀度** 会随时间增加，到达80%后会持续掉血，100%直接死亡
- **撤离过程中** 被攻击会中断撤离
- **死亡** 会失去所有携带的物品和装备

## 📝 Demo包含内容
