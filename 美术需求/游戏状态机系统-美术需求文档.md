## 一、需求概述

### 1.1 背景说明

游戏状态机系统是游戏的核心框架，负责管理游戏在不同阶段（启动、主菜单、家园、副本等）之间的切换。本需求文档列出该系统所需的所有美术资源。

### 1.2 整体风格

| 项目 | 说明 |
|------|------|
| **美术风格** | 暗黑幻想 + 科幻元素融合 |
| **色调** | 以深色为主，重点元素使用亮色（紫色、青色、金色）突出 |
| **UI风格** | 金属质感边框 + 半透明深色底 |
| **参考游戏** | 《暗黑地牢》《哈迪斯》《命运2》 |

### 1.3 技术规格

| 项目 | 规格 |
|------|------|
| 目标分辨率 | 1920 x 1080 |
| 图片格式 | PNG（带透明通道） |
| 九宫格切图 | 需提供切割参数 |
| 命名规范 | 小写字母+下划线，如 `btn_primary_normal.png` |

### 1.4 资源存放路径

