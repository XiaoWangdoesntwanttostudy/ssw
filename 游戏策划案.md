> **文档版本**：v5.0  
> **更新日期**：2025-12-04  
> **游戏类型**：世界重建 + 历史穿越 + 搜打撤

---

## 一、游戏概述

### 1.1 游戏名称
**《时空修复师》(Time Restorer)**

### 1.2 游戏类型
- 世界重建（类星露谷物语）
- 历史穿越（中国朝代副本）
- 撤离射击（类逃离塔科夫）
- 英雄养成（历史英雄）
- PvE 单机

### 1.3 目标平台
- PC（Windows）

### 1.4 核心概念

**玩家身份**：世界破碎后唯一的幸存者，肩负重建世界的使命

**核心设定**：
- 世界濒临破碎，最终只剩下主角和一方残存的异世界
- 异世界中有一台神秘的「时光机」，可以回溯到中国历史上的任意朝代
- 主角通过时光机穿越到各个朝代，搜刮重建世界所需的物资
- 带回的物资用于在异世界中逐步重建破碎的世界

**世界观**：
- 原本的世界因未知灾难而破碎，文明消亡
- 主角是唯一幸存者，被困在残存的异世界碎片中
- 时光机是上古文明遗留的神器，可以连接过去的时空
- 重建世界需要收集各个时代的「文明碎片」和物资

### 1.5 核心体验

| 体验 | 说明 |
|------|------|
| 世界重建 | 从废墟开始，逐步重建文明，恢复世界 |
| 历史穿越 | 穿越到中国各朝代，体验不同时代风貌 |
| 高风险探索 | 副本中搜打撤，活着回来才能保住战利品 |
| 英雄收集 | 在异世界发现历史英雄的灵魂，解锁使用 |
| 策略决策 | 去哪个朝代？派哪个英雄？何时撤离？ |

### 1.6 参考游戏

| 游戏 | 借鉴要素 |
|------|----------|
| 《星露谷物语》 | 家园建设、种植采集、经营节奏 |
| 《逃离塔科夫》 | 撤离机制、搜打撤玩法、物资风险 |
| 《暗黑地牢》 | 英雄管理、受伤冷却、永久风险 |
| 《哈迪斯》 | 英雄技能设计、战斗手感 |
| 《原神》 | 多角色切换、英雄养成深度 |
| 《文明》系列 | 历史文明元素、时代感 |

```
     ↓
```

### 2.3 核心决策点

| 决策 | 风险 | 收益 |
|------|------|------|
| 派强力英雄 vs 普通英雄 | 强力英雄受伤损失大 | 生存率高、效率高 |
| 带好装备 vs 轻装探索 | 好装备可能丢失 | 战斗力强 |
| 深入探索 vs 早点撤离 | 可能遇到强敌/时空崩塌 | 更多文明碎片 |
| 挑战朝代Boss vs 绕过去 | 可能失败 | Boss掉落传说物资 |

## 三、异世界·重建基地系统

### 3.1 基地概述

世界破碎后，主角被困在残存的异世界碎片中。这里是最后的避风港，也是重建世界的起点。玩家需要利用从历史朝代带回的物资和文明碎片，逐步重建破碎的世界。

**重建目标**：
- 收集足够的「文明碎片」修复世界
- 重建各种设施，恢复文明功能
- 最终目标：完全重建世界，恢复人类文明

### 3.2 基地功能

```
异世界·重建基地
├── 【时光机】（核心设施）
│   ├── 英雄选择/切换
│   ├── 朝代选择（副本入口）
│   └── 时空能量管理
├── 【迷雾区域】（可探索开拓）
│   ├── 开拓新区域获得建造空间
│   ├── 发现历史英雄的灵魂遗迹
│   └── 获取异世界特有资源
├── 【重建进度】
│   ├── 文明碎片收集进度
│   ├── 世界重建阶段
│   └── 解锁新功能/区域
├── 【住宅】
```

**迷雾地图机制**：
- 异世界初始只有基地核心区域可见，周围被混沌迷雾笼罩
- 玩家可以消耗资源开拓迷雾区域

## 四、时光机与英雄系统

### 4.1 时光机设定

时光机是上古文明遗留的神秘装置，可以连接过去的时空，让主角穿越到中国历史上的任意朝代。

**核心规则**：
- 时光机可以回溯到中国历史各朝代
- 每次穿越只能选择一个英雄形态
- 英雄受伤后需要冷却时间才能再次使用
- 英雄可以独立养成，拥有各自的等级和技能
- 穿越需要消耗「时空能量」，可通过副本获取

### 4.2 英雄解锁机制

**核心设定**：英雄是历史上的传奇人物，他们的灵魂被困在异世界的迷雾中。通过探索迷雾发现英雄遗迹，解锁后可以借用他们的力量穿越时空。

**未知英雄显示规则**：
| 状态 | 显示内容 |
|------|----------|
| 未发现 | 显示"？"，无任何信息 |
| 已发现未解锁 | 显示模糊立绘剪影，可见名称 |
| 已解锁 | 显示完整信息和立绘 |

**发现方式**：
- 探索异世界的迷雾区域，开拓新地图
- 在迷雾深处发现历史英雄的灵魂遗迹
- 完成特定条件（如击败守护者、收集碎片）后，英雄正式加入时光机

### 5.2 副本机制

#### 时空崩塌系统
```
时空崩塌机制：
├── 进入副本：崩塌度 = 0%
├── 时间流逝：每分钟 +5%
├── 特定区域：立即 +10-20%
└── 被时空异变体攻击：+5-10%

崩塌效果：
├── 0-25%：无影响
├── 25-50%：视野出现时空裂痕
├── 50-75%：属性下降10%
├── 75-100%：持续受到时空侵蚀伤害
└── 100%：强制传送回异世界（物资全丢）
```

#### 撤离系统
```
撤离方式：
├── 时空裂隙：地图标记位置，需要等待15-30秒完成回溯仪式
├── 紧急回溯：消耗时空信标，立即撤离但丢弃背包物资
└── Boss房撤离：击败朝代Boss后开启稳定传送门
```

## 六、战斗系统

### 6.1 战斗模式
- 第三人称视角（类塔科夫）
- 即时动作战斗
- 枪械+近战+技能结合

### 6.2 基础操作

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | WASD | 八方向移动 |
| 瞄准 | 鼠标右键 | 进入瞄准模式 |
| 射击/攻击 | 鼠标左键 | 普通攻击 |
| 技能1-4 | 1234 | 释放技能 |
| 闪避 | 空格 | 消耗耐力，有无敌帧 |
| 换弹/蓄力 | R | 换弹或蓄力攻击 |
| 交互 | E | 搜刮/开门/撤离 |
| 背包 | Tab | 打开背包 |

### 6.3 属性系统

| 属性 | 说明 |
|------|------|

## 十一、游戏流程

### 11.1 开局流程（剧情）

```
【序章：世界破碎】
1. 世界濒临破碎，主角目睹文明崩塌
2. 在最后时刻，主角被卷入时空裂隙
3. 醒来后发现自己身处残存的异世界碎片中
4. 发现上古遗留的时光机，了解重建世界的使命
5. 获得初始英雄「项羽」和「李广」的灵魂共鸣
6. 首次穿越到秦朝，学习搜打撤基础
7. 带回第一批文明碎片，开始重建之旅

【英雄发现流程】
- 探索异世界的迷雾区域，逐步开拓地图
- 在迷雾深处发现历史英雄的灵魂遗迹
- 英雄从"？"变为可见剪影，了解其身份
- 完成解锁条件后，英雄灵魂与主角共鸣，加入时光机
```

### 11.2 单局流程

```
异世界基地 → 时光机选择英雄 → 配置装备 → 选择朝代 → 穿越 →
    ↓
探索历史场景/战斗/搜刮文明碎片 → 找到时空裂隙 → 回溯仪式 →
    ↓
成功：结算界面 → 物资入库 → 返回异世界
失败：损失结算 → 英雄进入冷却 → 返回异世界
```

### 11.3 主线剧情

| 章节 | 目标 | 解锁朝代 |
|------|------|----------|
| 序章 | 了解世界观，学习基础 | 秦朝 |
| 第一章 | 收集500文明碎片 | 汉朝、三国 |
| 第二章 | 收集2000文明碎片 | 唐朝、宋朝 |
| 第三章 | 收集5000文明碎片 | 元朝、明朝 |
| 终章 | 收集10000文明碎片 | 清朝（终极副本） |
| 尾声 | 世界重建完成 | 新游戏+ |
