# 《深渊猎手》游戏策划案

> **文档版本**：v4.0  
> **更新日期**：2025-12-02  
> **游戏类型**：家园经营 + 撤离射击 + 英雄养成

---

## 一、游戏概述

### 1.1 游戏名称
**《深渊猎手》(Abyss Hunter)**

### 1.2 游戏类型
- 家园经营（类星露谷物语）
- 撤离射击（类逃离塔科夫）
- 英雄养成（多角色切换）
- PvE 单机

### 1.3 目标平台
- PC（Windows）

### 1.4 核心概念

**玩家身份**：一个被困在异空间的幸存者

**核心设定**：
- 玩家独自存在于一片神秘的异空间中
- 异空间中有一台「时光机」，可以让玩家幻化成不同的英雄
- 通过时光机进入各种危险副本，搜刮物资
- 带回的物资用于在异空间建设自己的家园

### 1.5 核心体验

| 体验 | 说明 |
|------|------|
| 家园建设 | 从零开始，一砖一瓦建造属于自己的家 |
| 高风险探索 | 副本中搜打撤，活着回来才能保住战利品 |
| 英雄收集养成 | 解锁、培养不同英雄，打造最强阵容 |
| 策略决策 | 带什么装备？派哪个英雄？何时撤离？ |

### 1.6 参考游戏

| 游戏 | 借鉴要素 |
|------|----------|
| 《星露谷物语》 | 家园建设、种植采集、经营节奏 |
| 《逃离塔科夫》 | 撤离机制、搜打撤玩法、物资风险 |
| 《暗黑地牢》 | 英雄管理、受伤冷却、永久风险 |
| 《哈迪斯》 | 英雄技能设计、战斗手感 |
| 《原神》 | 多角色切换、英雄养成深度 |

## 二、核心游戏循环

### 2.1 大循环

```
┌─────────────────────────────────────────────────────────────┐
│                    【异空间家园】                            │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐       │
│  │建造建筑  │  │种植采集  │  │制作装备  │  │整理仓库  │       │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘       │
│                       │                                     │
│                       ↓                                     │
│               ┌─────────────┐                              │
│               │   时光机    │                              │
│               │ (英雄选择)  │                              │
│               └──────┬──────┘                              │
└──────────────────────┼──────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────────┐
│                    【出击准备】                              │
│  选择英雄 → 配置装备 → 选择副本 → 进入                      │
└─────────────────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────────┐
│                 【副本探索】（搜打撤）                        │
│                                                             │
│    搜：探索地图，搜刮物资                                    │
│    打：与怪物战斗，击杀获取掉落                              │
│    撤：找到撤离点，完成撤离仪式                              │
│                       ↓                                     │
│    ┌─────────────────┴─────────────────┐                   │
│    ↓                                   ↓                   │
│  成功撤离                           撤离失败                │
│    ↓                                   ↓                   │
│  物资带回家园                       物资全部丢失            │
│  英雄获得经验                       英雄受伤（冷却中）       │
└─────────────────────────────────────────────────────────────┘
                       ↓
                  返回异空间家园
```

### 2.2 单局循环（搜-打-撤）

```
【进入副本】
     ↓
┌─────────────────────────────────────────┐
│ 【搜】探索阶段                           │
│  - 探索地图，开启宝箱                    │
│  - 搜刮物资、装备、材料                  │
│  - 发现隐藏区域                          │
└─────────────────────────────────────────┘
     ↓
┌─────────────────────────────────────────┐
│ 【打】战斗阶段                           │
│  - 第三人称射击/动作战斗                 │
│  - 使用英雄技能                          │
│  - 击杀怪物获得掉落                      │
└─────────────────────────────────────────┘
     ↓
┌─────────────────────────────────────────┐
│ 【撤】撤离阶段                           │
│  - 找到撤离点                            │
│  - 完成撤离仪式（可能被打断）            │
│  - 成功撤离 = 物资到手                   │
└─────────────────────────────────────────┘
```

### 2.3 核心决策点

| 决策 | 风险 | 收益 |
|------|------|------|
| 派强力英雄 vs 普通英雄 | 强力英雄受伤损失大 | 生存率高、效率高 |
| 带好装备 vs 轻装探索 | 好装备可能丢失 | 战斗力强 |
| 深入探索 vs 早点撤离 | 可能遇到强敌/时间耗尽 | 更多物资 |
| 挑战Boss vs 绕过去 | 可能失败 | Boss掉落顶级物资 |

## 三、异空间家园系统

### 3.1 家园概述

玩家独自存在于一片神秘的异空间，这里是安全的避风港。玩家需要利用从副本带回的物资，逐步建设自己的家园。

### 3.2 家园功能

```

├── 【时光机】

│   ├── 英雄养成界面
│   └── 副本入口
├── 【迷雾区域】（可探索开拓）
│   ├── 开拓新区域获得建造空间
│   ├── 发现隐藏的英雄遗迹
│   └── 获取异空间特有资源
├── 【住宅】
│   ├── 休息恢复
│   └── 存档点
├── 【仓库】
│   ├── 物资存储
│   └── 装备管理
├── 【工坊】
│   ├── 装备制作
│   ├── 道具制作
│   └── 装备强化
├── 【农场】
│   ├── 种植作物
│   └── 收获资源
├── 【矿场】
│   └── 采集矿石
├── 【伐木场】
│   └── 采集木材
└── 【其他设施】
    ├── 钓鱼点
    ├── 装饰物
    └── 扩展区域
```

### 3.3 异空间探索系统

**迷雾地图机制**：
- 异空间初始只有家园核心区域可见，周围被迷雾笼罩
- 玩家可以消耗资源开拓迷雾区域
- 开拓后的区域可用于建造、采集

**探索发现**：
| 发现类型 | 说明 |
|----------|------|
| 英雄遗迹 | 发现被困英雄的灵魂，完成条件后解锁英雄 |
| 资源点 | 新的矿脉、林地、水源等采集点 |
| 建造区域 | 扩展可建造的家园范围 |
| 神秘事件 | 随机事件，可能获得稀有物品或触发剧情 |

**开拓消耗**：
| 迷雾等级 | 消耗资源 | 可能发现 |
|----------|----------|----------|
| 浅层迷雾 | 木材x20 | 资源点、建造区域 |
| 中层迷雾 | 木材x50、石材x30 | 英雄遗迹、稀有资源 |
| 深层迷雾 | 魔法精华x10、铁锭x50 | 强力英雄、神秘事件 |

## 四、时光机与英雄系统

### 4.1 时光机设定

时光机是异空间中的神秘装置，允许玩家的意识幻化成不同的英雄形态，进入各个时空的副本中战斗。

**核心规则**：
- 每次出击只能选择一个英雄
- 英雄受伤后需要冷却时间才能再次使用
- 英雄可以独立养成，拥有各自的等级和技能

### 4.2 英雄解锁机制

**核心设定**：英雄通过探索异空间（家园所在的异世界）的迷雾地图发现，未解锁的英雄信息不可知。

**未知英雄显示规则**：
| 状态 | 显示内容 |
|------|----------|
| 未发现 | 显示"？"，无任何信息 |
| 已发现未解锁 | 显示模糊立绘剪影，可见名称 |
| 已解锁 | 显示完整信息和立绘 |

**发现方式**：
- 探索异空间的迷雾区域，开拓新地图
- 在迷雾深处发现被困的英雄灵魂/遗迹
- 完成特定条件（如击败守护者、收集碎片）后，英雄正式加入时光机

### 4.3 英雄列表

| 英雄 | 定位 | 武器类型 | 特点 | 发现地点（异空间迷雾区域） |
|------|------|----------|------|---------------------------|
| 猎人 | 远程输出 | 步枪/弓箭 | 平衡型，适合新手 | 初始英雄 |
| 战士 | 近战坦克 | 剑盾/大剑 | 高防御，近战强 | 初始英雄 |
| 刺客 | 近战爆发 | 匕首/短剑 | 高伤害，脆皮 | 迷雾森林·暗影小径 |
| 法师 | 远程AOE | 法杖 | 范围伤害，消耗大 | 迷雾高地·魔法石碑 |
| 医师 | 辅助生存 | 手枪 | 自我治疗，持久战 | 迷雾沼泽·废弃医疗站 |
| 工程师 | 道具流 | 冲锋枪 | 放置陷阱/炮台 | 迷雾废墟·机械残骸 |
| 狂战士 | 近战输出 | 双手斧 | 狂暴机制，高风险高回报 | 迷雾深渊·战魂祭坛 |
| 召唤师 | 召唤流 | 魔法书 | 召唤生物作战 | 迷雾核心·禁忌之地 |

### 4.4 英雄养成

```
英雄成长路径
├── 等级提升
│   ├── 副本中获得经验
│   ├── 升级获得属性点
│   └── 解锁新技能槽位
├── 技能树
│   ├── 主动技能（4个槽位）
│   ├── 被动技能（3个槽位）
│   └── 终极技能（1个槽位）
├── 装备系统
│   ├── 武器槽
│   ├── 防具槽（头/身/手/腿）
│   └── 饰品槽（2个）
├── 突破系统
│   ├── 消耗稀有材料
│   └── 提升等级上限
└── 好感度
    ├── 使用次数累积
    └── 解锁专属被动/皮肤
```

### 4.5 英雄状态

| 状态 | 说明 | 处理方式 |
|------|------|----------|
| 正常 | 可以出击 | - |
| 轻伤 | 属性-10% | 等待1小时恢复 |
| 重伤 | 无法出击 | 等待4小时或使用治疗道具 |
| 濒死 | 无法出击 | 等待24小时或使用稀有道具 |

### 5.2 副本机制

#### 侵蚀度系统
```
侵蚀度机制：
├── 进入副本：侵蚀度 = 0%
├── 时间流逝：每分钟 +5%
├── 特定区域：立即 +10-20%
└── 被特定怪物攻击：+5-10%

侵蚀效果：
├── 0-25%：无影响
├── 25-50%：视野减少
├── 50-75%：属性下降10%
├── 75-100%：持续掉血
└── 100%：强制传送回家园（物资全丢）
```

#### 撤离系统
```
撤离方式：
├── 固定撤离点：地图标记位置，需要等待15-30秒
├── 紧急撤离：消耗紧急信号弹，立即撤离但丢弃背包物资
└── Boss房撤离：击杀Boss后开启
```

### 5.3 失败惩罚

| 失败方式 | 物资损失 | 英雄状态 |
|----------|----------|----------|
| 被击杀 | 全部丢失 | 重伤（4小时冷却） |
| 侵蚀度100% | 全部丢失 | 轻伤（1小时冷却） |
| 紧急撤离 | 背包物资丢失，装备保留 | 正常 |

## 十一、游戏流程

### 11.1 开局流程

```
1. 玩家醒来，发现自己在神秘的异空间

3. 教程：基础操作、战斗、撤离
4. 首次副本探索，带回基础物资
5. 学习建造第一个建筑
6. 开始家园建设之旅

【英雄发现流程】
- 探索异空间的迷雾区域，逐步开拓地图
- 在迷雾深处发现英雄遗迹，英雄从"？"变为可见剪影
- 完成解锁条件（击败守护者/收集碎片）后，英雄加入时光机
```

### 11.2 单局流程

```
家园 → 时光机选择英雄 → 配置装备 → 选择副本 → 进入 →
    ↓
探索/战斗/搜刮 → 找到撤离点 → 撤离仪式 →
    ↓
成功：结算界面 → 物资入库 → 返回家园
失败：损失结算 → 英雄进入冷却 → 返回家园
```

## 附录：核心卖点

1. **家园+撤离的独特融合**：星露谷的温馨 + 塔科夫的紧张
2. **英雄切换系统**：不同英雄不同玩法，增加重玩性
3. **有意义的风险**：失败有代价，成功有成就感
4. **独立开发友好**：AI辅助 + Godot，成本可控
