# 《深渊猎手》- ARPG撤离养成游戏策划案

## 一、游戏概述

### 1.1 游戏名称
**《深渊猎手》(Abyss Hunter)**

### 1.2 游戏类型
- **ARPG + 撤离玩法**（Extraction ARPG）
- 搜刮探索 + 多职业养成
- Roguelite要素 + 持续经营
- PvE单机

### 1.3 目标平台
- PC（Windows）

### 1.4 游戏定位
玩家扮演一名"猎手公会"的管理者，招募并培养不同职业的猎手，派遣他们深入危险区域搜刮物资。成功撤离带回资源用于发展公会、培养更多猎手。**猎手死亡会丢失携带物资，且该角色永久死亡或需要高额复活费用**。

### 1.5 核心体验
- **多职业养成**：招募、培养不同职业的猎手，打造你的精英团队
- **高风险高回报**：派出的猎手可能回不来，但带回的物资极其丰厚
- **ARPG战斗**：爽快的动作战斗，技能连招，装备Build
- **策略经营**：发展公会、升级设施、管理资源
- **情感羁绊**：每个猎手都有故事，失去他们会心痛

### 1.6 参考游戏
| 游戏 | 借鉴要素 |
|------|----------|
| 《暗黑地牢》 | 角色管理、永久死亡、压力系统 |
| 《哈迪斯》 | ARPG战斗手感、技能设计 |
| 《逃离塔科夫》 | 撤离机制、物资风险 |
| 《暗黑破坏神》 | 装备系统、Build多样性 |
| 《火焰纹章》 | 角色羁绊、永久死亡的情感重量 |

### 2.2 单局循环（搜-打-撤）

```
【进入深渊区域】
     ↓
┌─────────────────────────────────────────┐
│ 【搜】探索阶段                           │
│  - 探索地图，开启宝箱、搜刮物资          │
│  - 寻找稀有材料和装备                    │
│  - 发现隐藏区域和秘密                    │

└─────────────────────────────────────────┘
     ↓ （遭遇怪物/主动挑战）
┌─────────────────────────────────────────┐
│ 【打】战斗阶段（ARPG）                   │
│  - 即时动作战斗                          │
│  - 使用技能连招                          │
│  - 击杀怪物获得掉落                      │
│  - 可选择规避强敌                        │
└─────────────────────────────────────────┘
     ↓ （决定撤离时机）
┌─────────────────────────────────────────┐
│ 【撤】撤离阶段                           │
│  - 找到/激活传送门                       │
│  - 完成撤离仪式（可能被打断）            │
│  - 成功撤离 = 物资到手 + 猎手存活        │
```

### 2.3 玩家身份

```
【玩家】= 公会管理者
    │
    ├── 在公会内：经营管理视角
    │   - 管理多个猎手
    │   - 升级设施
    │   - 分配资源
    │
    └── 出击时：操控选定的猎手
        - 第三人称/俯视角 ARPG 战斗
        - 一次只能操控一个猎手
        - 猎手死亡 = 真的死亡
```

### 2.4 核心决策点

| 决策 | 风险 | 收益 |
|------|------|------|
| 派精英猎手 vs 新手猎手 | 精英死了损失大 | 精英生存率高、收益高 |
| 带好装备 vs 轻装探索 | 好装备可能丢失 | 战斗力强、更安全 |
| 深入探索 vs 早点撤离 | 可能遇到强敌 | 更多物资、稀有掉落 |
| 挑战Boss vs 绕过去 | 可能团灭 | Boss掉落顶级物资 |

## 三、职业系统（核心）

### 3.1 职业总览

| 职业 | 定位 | 战斗风格 | 难度 |
|------|------|----------|------|
| **剑士** | 近战输出 | 刀剑连击、格挡反击 | ★★☆ |
| **狂战士** | 近战爆发 | 双手重武器、蓄力攻击 | ★★★ |
| **盗贼** | 刺客/敏捷 | 匕首、背刺、闪避 | ★★★ |
| **游侠** | 远程物理 | 弓箭、陷阱、走位 | ★★☆ |
| **法师** | 远程魔法 | 元素法术、AOE | ★★☆ |
| **牧师** | 辅助/生存 | 治疗、护盾、净化 | ★☆☆ |
| **召唤师** | 召唤流 | 召唤生物作战 | ★★★ |
| **炼金术士** | 道具流 | 投掷药剂、制造陷阱 | ★★★ |

### 3.2 职业详细设计

#### 【剑士】- 攻守兼备的近战职业
```
基础属性偏向：力量、体质
武器类型：单手剑+盾 / 双持剑

技能示例：
├── 三连斩：基础连击技能
├── 盾击：打断敌人攻击，造成眩晕
├── 剑刃风暴：旋转攻击周围敌人
├── 格挡反击：完美格挡后触发强力反击
└── 【终极】破甲斩：无视防御的致命一击
```

#### 【狂战士】- 高风险高伤害
```
基础属性偏向：力量
武器类型：双手巨剑/战锤

特殊机制：【狂怒值】
- 受伤和攻击积累狂怒
- 狂怒越高伤害越高，但受到伤害也越高

技能示例：
├── 重劈：高伤害单体攻击
├── 狂怒冲锋：冲向敌人造成击飞
├── 旋风斩：消耗狂怒的持续AOE
└── 【终极】狂暴化：短时间内攻击力翻倍
```

#### 【盗贼】- 高机动暗杀者
```
基础属性偏向：敏捷
武器类型：匕首、短剑

特殊机制：【潜行】【背刺加成】
- 脱战后可进入潜行
- 背后攻击造成额外伤害

技能示例：
├── 影步：短距离瞬移
├── 毒刃：攻击附带持续毒伤
├── 背刺：从背后攻击造成暴击
└── 【终极】暗影连杀：连续瞬移斩杀多个敌人
```

#### 【游侠】- 灵活的远程射手
```
基础属性偏向：敏捷、感知
武器类型：弓、弩

特殊机制：【蓄力射击】【标记猎物】
- 蓄力越久伤害越高
- 标记敌人后攻击有加成

技能示例：
├── 多重射击：一次射出多支箭
├── 穿透箭：贯穿多个敌人
├── 陷阱：放置减速/伤害陷阱
└── 【终极】箭雨：大范围AOE箭矢攻击
```

#### 【法师】- 强力的元素操控者
```
基础属性偏向：智力
武器类型：法杖、魔法书

特殊机制：【元素掌控】
- 可切换火/冰/雷三种元素
- 不同元素有不同效果

技能示例：
├── 火球术：基础远程伤害
├── 冰霜新星：周围敌人减速
├── 连锁闪电：在敌人间跳跃
├── 元素爆发：引爆元素状态
└── 【终极】陨石坠落：超大范围毁灭性攻击
```

#### 【牧师】- 团队守护者
```
基础属性偏向：智力、体质
武器类型：法杖、圣物

特殊机制：【神圣能量】
- 治疗和护盾消耗神圣能量
- 神圣能量随时间恢复

技能示例：
├── 治愈术：恢复生命值
├── 神圣护盾：吸收伤害的护盾
├── 净化：解除负面状态
├── 神圣惩击：对亡灵/恶魔特效
└── 【终极】神圣领域：大范围持续治疗+伤害减免
```

### 3.3 职业成长系统

```
猎手成长
├── 等级提升
│   ├── 获得属性点（自由分配）
│   └── 解锁新技能槽位
├── 技能升级
│   ├── 使用技能获得熟练度
│   └── 熟练度满可升级/进化技能
├── 职业精通
│   ├── 达到一定等级解锁进阶职业
│   └── 剑士 → 剑圣/圣骑士
└── 装备搭配
    └── 不同装备改变战斗风格
```

### 3.4 猎手招募

| 招募方式 | 说明 | 品质范围 |
|----------|------|----------|
| 普通招募 | 消耗金币，随机职业 | 白-蓝 |
| 高级招募 | 消耗稀有材料 | 蓝-紫 |
| 特殊招募 | 任务/活动获得 | 紫-金 |
| 培养新人 | 从低级培养到高级 | 取决于培养 |

**猎手品质**：影响初始属性和成长潜力
- ⚪ 普通（白）
- 🔵 精良（蓝）
- 🟣 稀有（紫）
- 🟡 传说（金）

## 四、物资与装备系统

### 4.1 物资分类

| 类别 | 说明 | 示例 |
|------|------|------|
| **武器** | 各职业专用武器 | 剑、弓、法杖、匕首 |
| **防具** | 护甲、头盔、护手等 | 布甲、皮甲、锁甲、板甲 |
| **饰品** | 戒指、项链、护符 | 增加属性或特殊效果 |
| **消耗品** | 药水、卷轴 | 治疗药水、传送卷轴 |
| **材料** | 制作/升级材料 | 矿石、皮革、魔法精华 |
| **贵重品** | 高价值物品（用于出售） | 宝石、古董、稀有素材 |
| **钥匙** | 开启特殊区域 | 地牢钥匙、宝箱钥匙 |
| **任务物品** | 特定任务需要 | 收集品、信物 |

### 4.2 装备品质与词缀

```
装备品质：
⚪ 普通（白）- 无词缀
🟢 魔法（绿）- 1个词缀
🔵 稀有（蓝）- 2个词缀
🟣 史诗（紫）- 3个词缀 + 特殊效果
🟡 传说（金）- 固定强力效果 + 套装潜力
🔴 神话（红）- 独一无二的顶级装备

词缀示例：
├── 前缀：锋利的（+攻击）、坚固的（+防御）
└── 后缀：of Fire（火焰伤害）、of Vitality（+生命）
```

### 4.3 物资风险机制

```
【出击时】
猎手装备 + 背包物资 ───→ 带入深渊
                              ↓
                    ┌─────────┴─────────┐
                    ↓                   ↓
               成功撤离              猎手死亡
                    ↓                   ↓
            物资带回公会          装备+物资全丢
            猎手获得经验          猎手死亡/重伤

【保险机制】
- 灵魂绑定：稀有装备可绑定，死亡不丢失（但需修复）
- 复活契约：提前购买，死亡后猎手可复活（消耗大量资源）
- 紧急传送石：危险时强制撤离（丢弃背包物资，保留装备）
```

### 4.4 背包系统

| 背包等级 | 格子数 | 获取方式 |
|----------|--------|----------|
| 布制背包 | 12格 | 初始 |
| 皮革背包 | 20格 | 制作/购买 |
| 冒险者背包 | 28格 | 稀有掉落 |
| 次元袋 | 40格 | 传说物品 |

- 不同物品占用不同格子
- 超重降低移动速度

### 5.2 区域机制

#### 深渊侵蚀
```
深渊侵蚀机制：
├── 进入深渊后，侵蚀度随时间上升
├── 侵蚀度影响：
│   ├── 25%：视野范围减少
│   ├── 50%：属性开始下降
│   ├── 75%：持续掉血
│   └── 100%：即死
└── 对策：
    ├── 净化药水：降低侵蚀度
    ├── 篝火休息点：重置侵蚀度
    └── 尽快撤离
```

#### 撤离点
```
撤离方式：
├── 传送门（固定位置）：稳定但可能被怪物占据
├── 召唤法阵（任意位置）：需要材料，施法时间长
├── 紧急传送石：立即撤离，但丢弃背包物资
└── Boss房撤离：击杀Boss后开启的特殊撤离
```

## 七、任务系统

### 7.1 任务类型

| 类型 | 说明 | 奖励 |
|------|------|------|
| 主线任务 | 推进剧情，解锁新内容 | 大量经验、解锁新区域/职业 |
| 日常委托 | 每日刷新的简单任务 | 金币、基础材料 |
| 周常挑战 | 高难度挑战任务 | 稀有材料、技能书 |

| 猎手专属 | 特定职业的挑战 | 职业专属装备/技能 |
| 成就任务 | 长期目标 | 称号、公会装饰、特殊奖励 |

### 7.2 任务示例

## 八、战斗系统（ARPG）

### 8.1 战斗基础

```
战斗操作
├── 普通攻击：连续点击进行连招
├── 技能：消耗MP/冷却时间
├── 闪避：消耗耐力，无敌帧
├── 格挡：减少伤害（部分职业）
├── 冲刺：快速移动
└── 道具：使用背包中的消耗品
```

### 8.2 属性系统

| 属性 | 说明 | 主要职业 |
|------|------|----------|
| 力量 | 物理攻击、负重 | 剑士、狂战士 |
| 敏捷 | 攻速、闪避、暴击 | 盗贼、游侠 |
| 智力 | 魔法攻击、MP | 法师、召唤师 |
| 体质 | 生命值、防御 | 全职业 |
| 精神 | MP回复、状态抗性 | 牧师、法师 |
| 幸运 | 掉落率、暴击伤害 | 全职业 |

### 8.3 伤害计算

```
伤害公式（简化）：
物理伤害 = (攻击力 - 防御力) × 技能倍率 × 暴击倍率
魔法伤害 = (魔攻 × 技能倍率) × (1 - 魔抗%) × 暴击倍率

暴击率 = 基础5% + 敏捷/200 + 装备加成
暴击伤害 = 150% + 幸运/100 + 装备加成
```

### 8.4 状态效果

| 状态 | 效果 | 来源 |
|------|------|------|
| 流血 | 持续物理伤害 | 利器攻击 |
| 中毒 | 持续伤害+减速 | 毒系攻击 |
| 燃烧 | 持续火焰伤害 | 火系法术 |
| 冰冻 | 无法移动 | 冰系法术 |
| 麻痹 | 无法行动 | 雷系法术 |
| 虚弱 | 攻击力下降 | 诅咒 |
| 破甲 | 防御力下降 | 特殊技能 |

### 8.5 元素克制

```
元素相克：
火 → 克制 → 冰 → 克制 → 雷 → 克制 → 火

物理 ←→ 魔法（互不克制）

克制效果：伤害+25%
被克制：伤害-25%
```

## 十一、游戏流程

### 11.1 新手引导

```
1. 序章
   - 剧情介绍（公会背景）
   - 获得第一个猎手（剑士）
   - 教程关卡（基础操作）
   - 首次撤离体验

2. 公会教程
   - 仓库使用
   - 装备猎手
   - 招募第二个猎手

3. 首次正式探索
   - 推荐去遗忘墓穴
   - 完成简单任务
   - 成功撤离奖励丰厚
   - 感受死亡惩罚（可选让一个弱猎手死亡）
```

### 11.2 单局流程

```
公会 → 选择猎手 → 装备配置 → 选择区域 → 出发 →
    ↓
加载进入深渊 → 探索/战斗/搜刮 → 找到撤离点 → 撤离仪式 →
    ↓
成功：结算界面（显示收获）→ 物资/经验结算 → 返回公会
失败：显示死亡原因 → 物资丢失 → 猎手状态（死亡/重伤）→ 返回公会
```

### 11.3 猎手死亡处理

```
猎手死亡后的选择：
├── 永久死亡（默认/困难模式）
│   └── 该猎手永久消失，装备全失
├── 复活（消耗大量资源）
│   ├── 消耗：复活药剂 + 金币
│   ├── 复活后：等级-1、需要休养数回合
│   └── 装备仍然丢失
└── 灵魂保存（特殊道具）
    └── 保留猎手，但等级降低更多
```

## 十二、经济与平衡

### 12.1 货币系统

| 货币 | 获取方式 | 用途 |
|------|----------|------|
| 金币 | 出售物资、任务 | 购买物资、招募、复活 |
| 灵魂碎片 | 击杀怪物 | 猎手升级、技能强化 |
| 公会徽章 | 主线任务、成就 | 解锁高级设施、特殊商品 |

### 12.2 平衡原则

- **前期节奏**：初期猎手弱，需要谨慎探索，学习游戏
- **死亡代价**：死亡有真实代价，但不会完全归零
- **成长曲线**：猎手养成需要时间，但可以感受到变强
- **职业平衡**：每个职业都有适合的场景和Build

### 12.3 防止滚雪球

- 高级区域消耗更多资源（侵蚀更快、药水消耗更大）
- 高等级猎手复活费用更高
- 稀有装备有耐久度，需要修复
- 探索有随机性，高配不一定稳赢

## 附录：快速参考

### 核心循环一句话
> 经营公会，培养猎手，派他们去深渊搜刮物资，活着回来就发财，死了就什么都没了。

### 游戏独特卖点
1. **ARPG + 撤离玩法**：不是射击，是爽快的动作战斗
2. **多猎手养成**：不是只养一个角色，是经营一支队伍
3. **真实的失去感**：猎手会死，死了是真的没了
4. **每局都有意义**：即使失败也推进了探索/收集

|------------|----------|
| 正面刚 | 剑士、狂战士 |
| 安全输出 | 游侠、法师 |
